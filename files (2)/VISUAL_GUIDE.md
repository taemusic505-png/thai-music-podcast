## 🎯 Data Structure

### Attendance Sheet Structure
```
┌────┬──────────┬──────────────┬────────────┬────────┐
│ #  │ Name     │ Subject      │ Date       │ Status │
├────┼──────────┼──────────────┼────────────┼────────┤
│ 1  │ Header   │ Header       │ Header     │ Header │
│ 2  │ สมชาย    │ คณิตศาสตร์   │ 2024-01-15 │ present│
│ 3  │ สมหญิง   │ คณิตศาสตร์   │ 2024-01-15 │ absent │
│ 4  │ สมทรง    │ ภาษาไทย      │ 2024-01-15 │ late   │
│ 5  │ สมชาย    │ ภาษาไทย      │ 2024-01-15 │ present│
└────┴──────────┴──────────────┴────────────┴────────┘

Status Values:
  - present: มาเรียนตามปกติ ✓
  - absent:  ขาดเรียน ✗
  - late:    มาช้ากว่าเวลา ⏰
```

### Assignments Sheet Structure
```
┌────┬──────────┬──────────────┬──────────────┬────────────┬──────────┐
│ #  │ Name     │ Subject      │ Assignment   │ DueDate    │ Status   │
├────┼──────────┼──────────────┼──────────────┼────────────┼──────────┤
│ 1  │ Header   │ Header       │ Header       │ Header     │ Header   │
│ 2  │ สมชาย    │ คณิตศาสตร์   │ การบ้านบท1   │ 2024-01-20 │ completed│
│ 3  │ สมหญิง   │ คณิตศาสตร์   │ การบ้านบท1   │ 2024-01-20 │ pending  │
│ 4  │ สมทรง    │ ภาษาไทย      │ สารเวชสาร   │ 2024-01-18 │ overdue  │
│ 5  │ สมชาย    │ ภาษาไทย      │ สารเวชสาร   │ 2024-01-18 │ completed│
└────┴──────────┴──────────────┴──────────────┴────────────┴──────────┘

Status Values:
  - completed: เสร็จแล้ว ✓
  - pending:   อยู่ระหว่างทำ ⏳
  - overdue:   เกินกำหนดส่ง ✗
```

## 🔐 Security Flow

```
┌─────────────┐
│  Webapp     │  (ใครก็อ่านได้ - ฟรี)
│  (Vercel)   │
└──────┬──────┘
       │ ใช้ API Key
       │ ที่ถูก restrict
       │ เฉพาะ domain นี้
       │
       ▼
┌─────────────────────────────────────┐
│  Google Sheets API                  │
│  (ที่ข้อมูล)                        │
│                                     │
│  API Key Restrictions:              │
│  ✓ HTTP referrers                   │
│  ✓ *.vercel.app                     │
│  ✓ yourschool.vercel.app            │
│                                     │
│  ✗ ห้าม access จากอื่น              │
└──────┬──────────────────────────────┘
       │
       ▼
┌─────────────────────────┐
│  Google Sheet (Private) │
│  (เก็บข้อมูลจริง)       │
│                         │
│ View: สาธารณะ          │
│ Edit: เฉพาะครู          │
└─────────────────────────┘
```

## 📊 ตัวอย่าง API Response

### เมื่อเรียก Attendance Data
```json
{
  "range": "Attendance!A1:D10",
  "majorDimension": "ROWS",
  "values": [
    ["Name", "Subject", "Date", "Status"],
    ["สมชาย", "คณิตศาสตร์", "2024-01-15", "present"],
    ["สมหญิง", "คณิตศาสตร์", "2024-01-15", "absent"],
    ["สมทรง", "คณิตศาสตร์", "2024-01-15", "late"]

}
```
